# 🧠 MathPulse

⚡ Train Faster. Think Sharper.

MathPulse is a modern mental math training platform with fast-paced challenges, streaks, analytics, XP levels, and beautiful themes designed to improve speed, focus, and calculation skills.

🌐 Open instantly in any browser — desktop, tablet, or mobile.

## ✨ Features

- ⚡ Mental Math Challenges
- 🧩 Multiple Difficulty Levels
- 🔥 Daily Streak System
- 📊 Performance Analytics
- 🎯 Accuracy Tracking
- 🎨 Theme Customization
- 🕹️ Gamified Experience
- 📱 Fully Responsive Design

## 🛠 Tech Stack

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion

## 🚀 Run Locally

```bash
npm install
npm run dev
